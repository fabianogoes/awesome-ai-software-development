# Proposito

Este repositorio existe para compartilhar conhecimento aberto sobre desenvolvimento de software com IA.

## O que o projeto busca fazer

- reunir ferramentas, frameworks, conceitos e referencias uteis
- publicar recursos compartilhaveis para workflows com agentes
- manter uma base aberta e pratica para quem quer estudar, comparar e aplicar essas abordagens

## O que o projeto evita

- acoplar a proposta a uma ferramenta especifica
- transformar a pagina publica em manual interno de manutencao
- expor detalhes operacionais que fazem sentido apenas para quem mantem este repositorio

## Principios

- conteudo publico deve ser util para a comunidade
- recursos compartilhados devem ser descritos de forma generica e agnostica
- documentacao interna e operacao do projeto ficam fora do site publico por padrao
